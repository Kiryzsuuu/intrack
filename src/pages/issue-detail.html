Description
Tulis unit test untuk auth middleware menggunakan Jest + Supertest. Test harus mencakup semua skenario berikut:

• Token valid → request diteruskan ke handler berikutnya
• Token expired → response 401 dengan pesan yang sesuai
• Token tidak ada / header kosong → response 401
• Token format salah (bukan Bearer) → response 401
• Token signature tidak valid → response 403

Pastikan coverage minimal 80% untuk file src/middleware/auth.middleware.ts. 
// contoh test case
describe('authMiddleware', () => {
  it('should pass valid token', async () => {
    const token = jwt.sign({ id: 1 }, process.env.JWT_SECRET);
    const res = await request(app)
      .get('/api/me')
      .set('Authorization', `Bearer ${token}`);
    expect(res.status).toBe(200);
  });
});
